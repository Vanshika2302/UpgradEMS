# UpGrad Virtual Event Management System

A full-stack Virtual Event Management System built using:

- ASP.NET Core Web API
- Angular
- SQL Server
- Bootstrap

---

# Features

## Admin Features

- Admin Login
- Manage Events
- Create / Update / Delete Events
- Manage Sessions
- Manage Speakers
- Assign Speakers to Sessions

## Participant Features

- User Registration
- User Login
- Explore Events
- Join Events
- View Registered Events
- Participant Dashboard

---

# Technologies Used

## Frontend

- Angular
- TypeScript
- Bootstrap
- HTML
- CSS

## Backend

- ASP.NET Core Web API
- Entity Framework Core
- SQL Server
